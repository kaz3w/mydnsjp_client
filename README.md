# mydnsjp_client
IP address update client script for MyDNS.JP

## Usage
usage: client.py [-h] [-r] [-v] [-d]

MyDNS.JP client

optional arguments:
  -h, --help     show this help message and exit
  -r, --reset    Reset address to 0.0.0.0
  -v, --verbose  Verbosity
  -d, --dryrun   Dry run(for checking)
